# MyMovieDB

## Overview
A TMDB-inspired frontend web app for discovering movies, TV shows, and actors. Built with React.js.

## Key Features
- Clean and modern UI
- Fully responsive layout built using CSS Grid
- Modular SCSS with mixins and variables
- Clean and maintainable codebase
- Search movies/TV/people
- Detailed info pages (plots, cast, ratings)

## Tech Stack
- Frontend: React.js, Axios, TMDB API, Tailwind 

## Installation
1. Clone repo: `git clone https://github.com/akashm01github/cineCloud.git`
2. Install deps: `npm install`
3. Run: `npm run start` (localhost:3000)

## Usage
- Browse popular content on homepage
- Search via top bar
